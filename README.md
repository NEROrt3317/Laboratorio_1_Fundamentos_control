# Laboratorio de MATLAB - Fundamentos de Control y Automatización

## Descripción

Este repositorio contiene el desarrollo del laboratorio realizado en MATLAB para la asignatura **Fundamentos de Control y Automatización**. El proyecto se enfoca en el análisis experimental y modelado matemático de un sistema térmico, utilizando datos reales obtenidos en laboratorio.

A partir de las mediciones registradas, se realizaron procesos de identificación de sistemas para obtener modelos dinámicos representativos, tales como:

- Modelo de Primer Orden con Tiempo Muerto (FOPDT)
- Modelo de Segundo Orden
- Comparación entre respuesta real y simulada
- Cálculo de parámetros característicos del sistema
- Evaluación del error entre modelos y datos experimentales

Además, se emplearon herramientas de MATLAB como:

- `tf()` para funciones de transferencia
- `step()` para simulaciones temporales
- `plot()` para análisis gráfico
- System Identification Toolbox
- Cálculo de métricas de error (ECM, RMSE)

## Objetivos

- Analizar la dinámica del sistema térmico.
- Ajustar modelos matemáticos a datos reales.
- Validar el comportamiento del modelo frente al sistema experimental.
- Aplicar conceptos fundamentales de control automático.

## Archivos incluidos

- Scripts `.m` de MATLAB
- Datos experimentales en Excel
- Gráficas comparativas
- Informe técnico del laboratorio

## Requisitos

- MATLAB R2022 o superior (recomendado)
- Control System Toolbox
- System Identification Toolbox
